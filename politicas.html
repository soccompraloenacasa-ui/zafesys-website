Sitio Web

Políticas para zafesys.com

Habeas Data

En cumplimiento de la Ley 1581 de 2012 y el Decreto 1377 de 2013 de Colombia, ZAFESYS garantiza la protección de tus datos personales. La información recolectada se utiliza exclusivamente para fines transaccionales, de envío y comunicación relacionada con tus pedidos.

Uso de la Información

Tus datos (nombre, dirección, teléfono, correo) son usados solo para procesar pedidos y garantizar la entrega.

No compartimos ni vendemos tu información a terceros.

Seguridad

Nuestro sitio web cuenta con certificados SSL y pasarelas de pago seguras para proteger toda la información transmitida.

Aplicación Móvil

Políticas para ZAFESYS Instaladores (Android)

Información General

ZAFESYS Instaladores es una aplicación móvil diseñada exclusivamente para los técnicos instaladores de cerraduras inteligentes de ZAFESYS. Esta aplicación permite gestionar las instalaciones asignadas, documentar el proceso y registrar la finalización de cada servicio.

Esta política describe qué información recopilamos a través de la aplicación, cómo la utilizamos y cómo la protegemos.

Datos que Recopilamos

La aplicación ZAFESYS Instaladores recopila los siguientes tipos de información:

Ubicación (GPS)

  • Qué recopilamos: Coordenadas GPS del dispositivo durante el horario laboral
  • Para qué: Verificar la llegada del técnico a la dirección de instalación y optimizar rutas de servicio
  • Cuándo: Solo cuando la aplicación está en uso activo

Fotos

  • Qué recopilamos: Fotografías tomadas antes y después de cada instalación
  • Para qué: Documentar el estado del área de instalación, evidencia de trabajo completado y protección ante reclamos
  • Almacenamiento: Las fotos se almacenan en servidores seguros de Cloudflare R2

Videos

  • Qué recopilamos: Videos cortos del proceso de instalación
  • Para qué: Documentación del servicio y capacitación
  • Almacenamiento: Los videos se almacenan en servidores seguros de Cloudflare R2

Firma Digital

  • Qué recopilamos: Firma digital del cliente al finalizar la instalación
  • Para qué: Confirmación de recibido y satisfacción del servicio
  • Almacenamiento: Las firmas se almacenan de forma segura asociadas a cada instalación

Información del Dispositivo

  • Qué recopilamos: Modelo del dispositivo, versión del sistema operativo, identificador único
  • Para qué: Diagnóstico de errores y mejora de la aplicación

Uso de la Información

La información recopilada se utiliza exclusivamente para:

  • Gestionar y asignar instalaciones a técnicos
  • Verificar la realización correcta de los servicios
  • Documentar el antes y después de cada instalación
  • Proteger tanto al cliente como a la empresa ante posibles disputas
  • Calcular tiempos de instalación y optimizar procesos
  • Mejorar la calidad del servicio

No vendemos, alquilamos ni compartimos información personal con terceros para fines de marketing.

Permisos de la Aplicación

La aplicación solicita los siguientes permisos del dispositivo:

  • Cámara: Para tomar fotos y videos de las instalaciones
  • Ubicación: Para registrar la llegada al sitio de instalación
  • Almacenamiento: Para guardar temporalmente archivos antes de subirlos
  • Internet: Para sincronizar datos con el servidor

Todos los permisos son necesarios para el funcionamiento correcto de la aplicación y solo se activan cuando el usuario los autoriza.

Seguridad de los Datos

Implementamos medidas de seguridad para proteger tu información:

  • Transmisión de datos mediante conexiones HTTPS cifradas
  • Almacenamiento en servidores seguros con acceso restringido
  • Autenticación de usuarios para acceder a la aplicación
  • Copias de seguridad regulares

Retención de Datos

Los datos de las instalaciones (fotos, videos, firmas) se conservan por un período de 5 años para cumplir con obligaciones legales y de garantía. Después de este período, los datos son eliminados de forma segura.

Los datos de ubicación se conservan por 90 días para fines operativos.

Derechos del Usuario

De acuerdo con la Ley 1581 de 2012, tienes derecho a:

  • Conocer: Acceder a tus datos personales almacenados
  • Actualizar: Corregir información inexacta o incompleta
  • Revocar: Solicitar la eliminación de tus datos cuando ya no sean necesarios
  • Presentar quejas: Ante la Superintendencia de Industria y Comercio (SIC)

Para ejercer estos derechos, contáctanos a través de los canales indicados abajo.

Cambios en esta Política

Podemos actualizar esta política de privacidad ocasionalmente. Notificaremos cualquier cambio significativo a través de la aplicación o por correo electrónico. Te recomendamos revisar esta página periódicamente.
